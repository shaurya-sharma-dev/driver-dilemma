# Driver Dilemma

Driver Dilemma is an open-source driving game built in Python using Ursina, with Blockbench for modeling and texturing. It is licensed under the MIT license.

## Controls
- WASD/Arrow Keys - Player Movement
- R - Reset
- Escape - Open Pause Menu

## Winner Area
<p>Once all levels have been completed, the player is taken to a "winner area" where there are no obstacles. Going into the finish line in the winner area will send the player back to level 1.</p>
